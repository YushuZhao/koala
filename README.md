# Tripod Template

## 项目结构

+ `__test__`: 单元测试等
+ `.husky`: husky配置
+ `config`: Webpack-chain 配置
+ `lib`: Nodejs 帮助函数
+ `public`: 静态文件
+ `script`: 启动脚本
+ `src`: 项目代码
+ `.babelrc`: babel 配置
+ `.editorconfig`: editor 配置文件 配合 EditorConfig for VS Code 使用
+ `.env*`: 环境变量配置 配合 dotenv 使用
+ `.eslint*`: eslint 相关
+ `.gitignore`: git 忽略文件
